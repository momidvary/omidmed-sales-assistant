# دستیار فروش امیدمِد — مرحله اول

این نسخه فقط اسکلت اولیه و داشبورد پروژه است. هنوز دیتابیس، ورود اکسل هلو و هوش مصنوعی فعال نشده‌اند.

## اجرا در ویندوز

PowerShell را باز کنید و وارد پوشه پروژه شوید:

```powershell
cd C:\Projects\omidmed-sales-assistant
npm install
npm run dev
```

سپس آدرس زیر را در مرورگر باز کنید:

```text
http://localhost:3000
```

## بررسی نهایی

```powershell
npm run lint
npm run build
```
